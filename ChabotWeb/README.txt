About:
This chatbot is created using AIML(Artificial Intelligence markup language) 
and the python library Pyaiml which supports it.
For web interfaces used Django framework <MVC>.

Note:
This Application is not production ready.It is just an experimental one.

Requirements:
Python 2.7
Django 1.11
aiml 0.8.6
AIML
